
# Color Palette Generator

A beautiful and interactive Color Palette Generator built with HTML, CSS, and Vanilla JavaScript. This application allows users to generate random color palettes, adjust hue, saturation, and brightness, lock specific colors, and save their favorite palettes to a local library.

## Features

-   **Random Palette Generation**: Instantly generate a set of 5 random colors.
-   **Color Adjustments**: Fine-tune each color using Hue, Brightness, and Saturation sliders.
-   **Lock Colors**: Lock your favorite colors and generate new ones for the remaining unlocked slots.
-   **Copy to Clipboard**: Click on the Hex code to copy the color to your clipboard.
-   **Save Palettes**: Save your custom palettes with a name.
-   **Library**: Access your saved palettes from the library and reload them.
-   **Local Storage**: Palettes are saved to the browser's local storage, so they persist after refresh.

## Technologies Used

-   **HTML5**: Structure of the application.
-   **CSS3**: Styling and animations.
-   **JavaScript (ES6+)**: Application logic and DOM manipulation.
-   **Chroma.js**: A small-ish zero-dependency JavaScript library for all kinds of color conversions and color scales.
-   **Boxicons**: For the icons used in the UI.
-   **Google Fonts**: "Mulish" font for typography.

## Setup & Usage

1.  Clone the repository or download the source code.
2.  Open `index.html` in your web browser.
3.  **Generate**: Click the "Generate" icon to get new colors.
4.  **Adjust**: Click the sliders icon on any color to open adjustment controls.
5.  **Lock**: Click the lock icon to keep a color while generating others.
6.  **Save**: Click the save icon, enter a name, and save your palette.
7.  **Library**: Click the book icon to view and load saved palettes.
